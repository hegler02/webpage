# Knowledge Blocks Lecture 30

학교 규정집을 업무별 지식파일과 민원봇/직원봇 샘플로 바꾸는 30장 강의 슬라이드입니다.

## 파일 구조

- `index.html`: 강의 슬라이드 화면 골격
- `styles.css`: Knowledge Blocks 템플릿 스타일
- `slides.js`: 30장 강의 내용과 블록 좌표
- `deck.js`: 렌더링, 전환, 키보드, 버튼, 스와이프 동작

## 실행

`http://127.0.0.1:4189/knowledge_blocks_lecture_30/index.html`

## 수정 원칙

- 강의 문구는 우선 `slides.js`만 수정합니다.
- 템플릿 원본 `knowledge_blocks_template`은 보존합니다.
- 한 장에는 하나의 핵심 메시지만 둡니다.
